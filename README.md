# invenity-labs/homebrew-tap

Homebrew tap for Invenity Labs LLC's open-source tools.

## Usage

```sh
brew tap invenity-labs/tap
```

Then install whichever formula you want, e.g.:

```sh
brew install go-seccure
```

## Formulas

| Name | Description | Source |
|---|---|---|
| `go-seccure` | Pure-Go port of SECCURE, bit-for-bit wire-compatible with the C reference and py-seccure. | [invenity-labs/go-seccure](https://github.com/invenity-labs/go-seccure) |

Formula files are auto-generated and pushed by each upstream project's
[goreleaser](https://goreleaser.com) release pipeline; do not edit by hand.
